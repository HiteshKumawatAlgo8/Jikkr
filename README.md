# Editorial Standalone Shopify Theme

This ZIP is a **standalone Shopify theme**, not a section-only bundle.

## What is included
- Complete `layout/theme.liquid`
- Header + footer
- Homepage with the animated editorial scroll section
- Product page
- Collection page
- Cart page
- Search page
- Blog + article pages
- Contact page template
- Basic customer account templates
- Theme settings and editable section settings

## Upload steps
1. In Shopify admin go to **Online Store → Themes**.
2. Click **Add theme → Upload zip file**.
3. Upload `shopify_editorial_standalone_theme.zip`.
4. After upload, open **Customize**.
5. Add your images to the homepage editorial section.
6. Set your header menu, footer menu, logo, contact details, and featured collection.

## Recommended post-upload edits
- Replace default placeholder images in the editorial section
- Set a main menu and footer menu
- Create a collection and connect it to the Featured Collection section
- Create a page and assign the **contact** template if needed
- Review mobile spacing after adding final media

## Notes
- The animated homepage is the custom visual centerpiece.
- Cart, product, collection, search, page, article, and blog templates are included.
- Checkout remains Shopify-controlled, as with all normal Shopify themes.
